# Pet-life

A new Flutter project about pets.

## Getting Started

starting command:

- open -a Simulator
- flutter run

## Backend
* [Pet](https://github.com/abserari/pet): The go http server provide API we need.
      Get your own mysql database url link and write it to this server.
      
      go into main and run
      go run ./main.go
